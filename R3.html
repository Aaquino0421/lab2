I’m slowly learning and adapting to html codes. So far, I find it interesting and enjoying at the same time. My knowledge in IT is slowly widening because of the professor’s help. This might be a big challenge for me, but I do believe that I can be able to learn more and possibly be able to master other codes.

Throughout the process, I found out that Sublime software is easier to understand than doing html application using notepad. I like how user friendly the software is especially for beginners like me. Creating a repository using Github and was very convenient and useful it has features where you can view,edit, and delete repositories anytime and anywhere. I think Github and Heroku are revolutionary software not only because they can store huge amounts of data and does not require to bring external storage devices but also, because of its convenience to its users. I think that the creator of this software does not only think about the importance of the software to its user but also, they thought about the convenience of their software to its user especially nowadays not all people works in an office, some are freelancers where they need an immediate place to edit document and that I think is the best part of this software.
